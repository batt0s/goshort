How to use API?

You can use the API in your apps freely.

shorten

POST {{ .Host }}/api/shorten

*url : string

shorten/custom

POST {{ .Host }}/api/shorten/custom

*url : string, *custom : string

getOrigin

POST {{ .Host }}/api/getOrigin

*url : string

Examples :

Curl

            # Shorten URL :
            curl -X POST 'http://{{ .Host }}/api/shorten' -H 'Content-Type: application/json' -d '{"url":"URL to short"}' 
            # Shorten with custom short URL :
            curl -X POST 'http://{{ .Host }}/api/shorten/custom' -H 'Content-Type: application/json' -d '{"url":"URL to short","custom":"Custom Short URL"}' 
            # Get Original URL : 
            curl -X POST 'http://{{ .Host }}/api/getOrigin' -H 'Content-Type: application/json' -d '{"url":"Short URL"}'

Pyhton

            import requests
            url = "http://{{ .Host }}/api/"
            # Shorten URL :
            res = request.post(url+"shorten", json={'url':'URL to short'})
            print(res.text)
            # Shorten URL with custom short URL :
            res = request.post(url+"shorten/custom", json={'url':'URL to short','custom':'Custom short URL'})
            print(res.text)
            # Get Original URL :
            res = request.post(url+"getOrigin", json={'url':'Short URL'})
            print(res.text)
